EasyAccept

Description

Runs EasyAccept given the facade to be used and paths to the script files.

Note: This task depends on external libraries not included in the Ant distribution.

Parameters

Attribute Description Required
facade The Facade class full qualified name. yes
failonerror Fails build process in case of error. A test failure is not considered an error. no
failureproperty The name of the property where a "true" value will be stored in case of test failure. no

Parameters specified as nested elements

path

The path where the script files will be.

Examples

    <easyaccept facade="test.Facade" failureproperty="test.failed">
        <path>
            <fileset dir="${basedir}/test/script*.txt"/>
        <path>
    </easyaccept>
Runs EasyAccept with test.Facade as facade class to be used and all files in ${basedir}/test directory that matches to script*txt regex. Case any of the tests fails, true will be stored in ${test.failed} property.
